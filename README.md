# Tanssitietokanta

This is a software used to organize and display dance instructions for historical dance balls.

It has been used in some form in Ropecon to plan balls and show instruction slides since 2016.

## Features

* A dance database for storing dances, their instructions, lengths and other data
* Integration to the [Finnish historical dance wiki](https://tanssi.dy.fi/) for fetching dance instructions
* Ability to make events with workshops and a ball with dances
* The ball program has a slideshow for showing dance instructions and other info
* Ability to generate dance instruction leaflets for printing
* Ability to genereta checkbox style cheat lists for marking which dances you have learned
* Real-time collaborative editing

## Project Structure

Tanssitietokanta is organized into two typescript projects in their respective
folders: [frontend](frontend) and [backend](backend).
See their respective readme's for installation and operation instructions.

There is also a legacy version in the legacy folder, but it is no longer developed.
